## Ce que ce mod traduit

| Catégorie | Contenu | Volume |
|-----------|---------|--------|
| Dialogues | Rencontres, négociations, quêtes, événements | 4 933 textes |
| Interface | Conseils, tooltips, notifications Intel | 560+ textes |
| Codex | Descriptions de vaisseaux, armes, systèmes | 570 entrées |
| Vaisseaux | Noms, désignations, variantes | 2 187+ noms |
| Équipement | Armes, modifications de coque, systèmes | 360+ entrées |
| Campagne | Marchés, industries, marchandises, compétences | 280+ entrées |
| Factions | Grades, flottes, entités, planètes | 7 fichiers |
| Missions | Briefings de combat | 15 missions |

> Compatible avec tous les mods — ne remplace que les fichiers de traduction vanilla.

---

## Installation

1. [Téléchargez la dernière version](https://github.com/mipsou/starsector_lang_pack_fr/releases/latest)
2. Extrayez l'archive dans le dossier `mods/` de Starsector
3. Activez **Starsector Language Pack - French** dans le launcher

## Limitations connues

Certains textes sont codés en dur dans le moteur Java de Starsector et ne peuvent pas être traduits par un mod de traduction :

- Interface de dialogue ("You decide to...", "Open a comm link")
- Certaines variables résolues en anglais par le moteur
- Les guillemets français « » ne s'affichent pas (limitation de la police du jeu)

Ces limitations seront traitées dans une future version avec un override Java.
