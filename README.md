### Installation

1. Téléchargez la [dernière version](https://github.com/mipsou/starsector_lang_pack_fr/releases/latest)
2. Extrayez l'archive dans le dossier `mods` de Starsector
3. Activez le mod dans le launcher du jeu

### Compatibilité

- Starsector : 0.97a-RC11
- Compatible avec tous les mods (ne modifie que les fichiers de traduction)

### Notes de version

- **v1.2.7** : rules.csv — 4 933 dialogues complets en français (rencontres, factions, quêtes, événements). 15 missions traduites.

### Limitations connues

- Certains textes d'interface restent en anglais (hardcodés dans le moteur Java : "You decide to...", "Open a comm link", "Admiral skills:", etc.)
- Les variables Java (`$shipOrFleet`, `$rank`) sont résolues en anglais par le moteur
- Le mot "Military" dans les sous-marchés est généré par le moteur
- Les guillemets français « » ne sont pas rendus par la police du jeu (affichés comme ")

### Roadmap

- **v1.3+** : Override Java pour les textes hardcodés (UI, variables)
- **Futur** : Audit qualité des traductions existantes
