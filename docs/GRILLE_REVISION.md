# Grille de Révision des Traductions

## Critères d'Évaluation

### 1. Cohérence Terminologique
- [ ] Termes du jeu cohérents
- [ ] Noms propres respectés
- [ ] Abréviations standardisées
- [ ] Variables système préservées

### 2. Qualité Linguistique
- [ ] Orthographe correcte
- [ ] Grammaire respectée
- [ ] Ponctuation appropriée
- [ ] Style fluide

### 3. Adaptation Culturelle
- [ ] Expressions adaptées
- [ ] Références culturelles ajustées
- [ ] Ton approprié au contexte
- [ ] Humour préservé si pertinent

### 4. Format Technique
- [ ] Encodage UTF-8
- [ ] Structure JSON/CSV respectée
- [ ] Variables intactes
- [ ] Pas de caractères spéciaux invalides

### 5. Contexte du Jeu
- [ ] Cohérence avec l'univers
- [ ] Terminologie spécifique
- [ ] Ton narratif adapté
- [ ] Immersion préservée
